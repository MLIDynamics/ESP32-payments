Maksudemo

Tämän demosivun avulla voit testata maksutapahtuman.

  1. Pistä ESP32:seen virrat päälle
  2. Paina "Pyydä Maksutapahtuma"
  3. Sivu esittää sinulle maksuosoitteen - ohita se painamalla "Esitä maksavasi"
  4. Katso, kuinka ESP32 indikoi maksun tapahtuneen
